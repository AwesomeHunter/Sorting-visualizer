# Sorting visualizer
Simple sorting visualizer.
## Controls:
* `Scroll` Change speed of animation
* `=` Increase data size
* `-` Decrease data size
* `s` Shuffle data
* `t` Stop algorithm
* `1` Run bubble sort
* `2` Run selection sort
* `3` Run insertion sort
* `4` Run merge sort
## Dependencies:
* SFML 2.5.1
